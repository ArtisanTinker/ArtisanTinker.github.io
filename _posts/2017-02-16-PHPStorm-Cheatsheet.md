## Top Ones

**ALT+Home** - Breadcrumbs Nav

## The Rest

**ALT F1 1** - Navigate to Project Explorer

**CTRL E** - Open recent

**ALT click sidebar** - Show error

**CTRL + SHIFT + Backspace**  - Reformat code

**CTRL + Space** -reopen code window
